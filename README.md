# Installation
1. Clone repository following this command
```
https://github.com/aliefabdillah/be-investKuy-app.git
```
2. Install dependencies
```
npm install
```
3. run application 
```
npm run start-dev
```

# Setup .env file
- rename file .env.aws to .env
